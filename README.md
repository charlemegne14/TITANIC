# 🚢 TITANIC - Analyse des Facteurs de Survie et Profilage des Passagers

Projet classique d’analyse de données sur le jeu de données **Titanic**, célèbre pour l’étude des facteurs influençant la survie.

## 🎯 Objectifs du Projet
- Explorer les données des passagers (âge, classe, sexe, famille, etc.)
- Identifier les facteurs clés de survie
- Réaliser le profilage des passagers
- Produire des analyses, visualisations et une présentation

## 🛠️ Solutions Mises en Œuvre
- Nettoyage et analyse statistique des données
- Calculs de taux de survie par catégorie
- Création de visualisations claires
- Préparation d’une présentation des résultats

## 📁 Structure du Projet

| Fichier               | Description |
|-----------------------|-----------|
| `titanic.csv`         | Jeu de données principal |
| `Analyses.xlsx`       | Analyses statistiques détaillées |
| `Presentation.pptx`   | Support de présentation |
| `questions .docx`     | Questions et réponses du projet |

## 🛠️ Technologies
- **Microsoft Excel**
- **PowerPoint**
